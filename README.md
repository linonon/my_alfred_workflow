# Alfred Workflow

这个项目放在自己的 `~` 目录下, 保证在不同电脑上逻辑一致

需要安装:

- [Alfred](https://www.alfredapp.com/)
- [zoxide](https://github.com/ajeetdsouza/zoxide): `brew install zoxide`
